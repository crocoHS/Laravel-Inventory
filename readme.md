# APEX

Apex is a self hosted sales, purchase, inventory and accounting solution for small businesses.

## Work in progress

Apex is work in progress and misses several key features :

- Sales
    - [ ] Client
    - [ ] Quotation
    - [ ] Sales Order
    - [ ] Invoice
    - [ ] Payment Received

- Inventory
    - [ ] Items
    - [ ] Receive
    - [ ] Delivery Note
    - [ ] Journal

- Purchase
    - [ ] Vendor
    - [ ] Expense
    - [ ] Purchase Order
    - [ ] Bill
    - [ ] Payment Made

- Accounting
    - [ ] Statement of Account
    - and more...
